## Phase Goal (User Story)

**As a** 終端機使用者（記帳者），**I want to** 用四個指令（add / list / total / rm）新增、列出、加總、刪除支出並持久化到 JSON 檔，**so that** 我能在不裝任何第三方套件的情況下，可靠地管理自己的花費，且行為由全綠測試套件保證。

> 註：ROADMAP `**Goal:**` 為敘述句而非嚴格 user-story 格式；此故事由該 Goal 直接推導（意圖明確、無新增範圍）。此為 CLI 工具，無 HTTP/路由/瀏覽器/部署層 —— walking-skeleton 的「最薄端到端切片」對應為：一支可執行的 `expense.py`，完成一次真實 JSON 讀寫 + 至少一個經 argparse 分派的指令 + 一個通過的 pytest 測試。
